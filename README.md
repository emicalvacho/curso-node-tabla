# Notas

### Command

```
node app [options]
```

### Options

```
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     is the base to multiply                     [number] [required]
  -l, --list     show or not list                     [boolean] [default: false]
  -h, --height   limit amount number to multiply          [number] [default: 10]
```
